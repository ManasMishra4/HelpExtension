Code Helper AI

This is a custom browser extension to help with coding problems.

To use the extension, follow these steps:

  1. Download the zip file below.
  2. Extract the files anywhere you want and name the folder `CodeHelp`.
  3. Open your browser, go to `Extensions`, and click `Load Unpacked`.
  4. Select the `CodeHelp` folder you just extracted.
  5. Enjoy your extension!
